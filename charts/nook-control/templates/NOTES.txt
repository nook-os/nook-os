1. Point that DNS name at the agent Service's external address:

     kubectl -n {{ .Release.Namespace }} get svc {{ include "nook-control.componentName" (dict "root" . "component" "agent") }} \
       -o jsonpath='{.status.loadBalancer.ingress[0].ip}{"\n"}'

2. Read the listener cert's fingerprint (pins the self-signed cert on each node):

     kubectl -n {{ .Release.Namespace }} get secret {{ .Values.agent.tlsSecret }} \
       -o jsonpath='{.data.{{ .Values.agent.tlsCertKey }}}' | base64 -d \
       | openssl x509 -outform der | sha256sum | cut -d' ' -f1

3. On each node, enrol with a join token from the UI and that fingerprint:

     nook enroll --server https://{{ .Values.agent.publicUrl }} \
       --token <join token> --server-fingerprint <fingerprint>
